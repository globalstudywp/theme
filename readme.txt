== Description ==

GlobalStudy is a starter theme from MasterStudy LMS plugin provides a complete and ready-made site template for eLearning and online education websites.

The theme is fully responsive with a cutting-edge design that can be customized according to the user's requirements. Packed with Elementor, WooCommerce, and H5P plugins, making it an all-in-one solution to build a professional website for creating and selling courses online.

GlobalStudy natively works with the MasterStudy LMS plugin and helps to customize the header, footer, and menus to match the specific brand image. The theme is designed to be SEO-friendly, ensuring that the website ranks higher on search engines.

This can help attract more traffic to the eLearning website and increase engagement. A number of exquisite LMS widgets: Courses Carousel, Courses Grid, Search Box, Featured Teacher, and Recent Courses speed up the running full-fledged education website. A simple yet efficient demo importer will allow you to easily install the entire layout that will fit perfectly on your WordPress.

== Installation ==
= From within WordPress =
1. Visit "Appearance > Themes > Add New"
2. Search for "GlobalStudy"
3. Install and activate
4. Go to GlobalStudy Admin page
5. Click to the Demo Import button
